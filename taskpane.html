CONSTRUCT Function

AI-powered text generation for Excel

How to use:

Use the CONSTRUCT function in any cell:

=CONSTRUCT.CONSTRUCT("user prompt", "system prompt")

Basic Example:

=CONSTRUCT.CONSTRUCT("What is AI?", "You are a helpful assistant")

With Parameters:

=CONSTRUCT.CONSTRUCT("Explain quantum computing", "You are a physics professor", 0.5, "direct", 500, "gpt-4")

Parameters:

  • userPrompt (required): Your question or prompt
  • systemPrompt (required): Instructions for the AI
  • temperature (optional): 0-2, controls randomness
  • mode (optional): "direct", "model", or "optative_rag"
  • maxTokens (optional): 1-4096, response length limit
  • llmAlias (optional): Model to use
